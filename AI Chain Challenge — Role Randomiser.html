Game Mechanics, Tagging & Scoring tap to collapse
0–10 min
Ideation Sprint + Hero Idea Submission
Prompt Engineers generate AI use cases. Architect consolidates. DevOps submits continuously using the correct tag format.
  • Speed Bonus — first team to 5 valid submissions wins +15 pts (auto-tracked)
  • Developer identifies the Hero Idea — must be Agentic AI linking 2+ departments
  • DevOps submits Hero Idea with the [HERO] tag before minute 10
  • No tag = no score. Missing or wrong format means the entry is disqualified
[HERO] tag submitted after min 10 is not eligible for the Power Up
10–12 min
Leaderboard Revealed — Review Window
All tagged submissions become visible on the shared leaderboard. Sprint is closed — no new original ideas accepted.
  • Teams study rival submissions and identify ideas worth stealing
  • Developer picks which ideas can be genuinely improved
  • Facilitator announces Steal window opens at minute 12
12–15 min
Steal Window — Improve to Score
Teams steal up to 3 rival ideas and resubmit improved versions. Each steal uses one Steal Power Up Card.
  • Must use the [STOLEN from Team Name] tag — missing tag = no ×3, scored as regular
  • Claude judges: genuinely better = ×3 tier score
  • Claude determines not better = 0 score for that entry
  • Maximum 3 steals per team — one per Steal Power Up Card
15+ min
Claude Scores All Submissions
Claude Operator pastes all entries. Claude validates tags, assigns tiers, applies Power Ups, and produces the final leaderboard.
  • Entries missing required tags are automatically disqualified — 0 points
  • Hero Idea outcome applied — confirmed ×2 / rejected −5 / not submitted −10
  • Each steal judged — improved ×3 / not improved 0
  • Final leaderboard posted · Winner announced · Golden Idea revealed
Golden Rule: No proper tagging = no points. No exceptions, no appeals.
The DevOps Engineer applies every tag. The Architect checks format before passing entries to DevOps.
Regular Idea
Standard entry · scored by tier
[Team Name]  idea description
Example
[Team Bandwidth] AI-powered propensity scoring model that analyses real-time behavioural signals to identify high-intent prospects and trigger personalised offers automatically
Team name tag is mandatory — missing tag = 0 points, no exceptions
Use your exact assigned team name as shown in the role assignment
Tag must appear at the very start of the entry, before the idea description
Hero Idea
Must submit before minute 10
[Team Name]   [HERO]  Agentic AI idea spanning 2+ departments
Example
[Team Bandwidth] [HERO] Agentic AI system that monitors network anomaly signals in real time, autonomously correlates faults to affected customers, triggers personalised outage notifications, and schedules technician dispatch — all without human intervention, spanning Network Operations and Customer Experience
[HERO] tag must appear after the team name — without it the entry scores as a regular idea only, no Power Up applied
The idea itself must describe Agentic AI behaviour and reference at least 2 departments — Claude checks this at scoring time
Must be submitted before minute 10 — late entries lose the Power Up but are still scored as regular ideas
One Hero Idea per team — if multiple [HERO] entries are submitted, only the first is evaluated
Stolen Idea
Steal window only · min 12–15
[Team Name]   [STOLEN from Original Team Name]  improved idea description
Example
[Team Bandwidth] [STOLEN from Team Latency] Improved: AI billing anomaly engine that not only flags invoice errors before they are issued, but autonomously traces the root cause in the rating engine, corrects affected records, and reprocesses billing without human intervention — cutting the current 23-day revenue recognition lag to under 48 hours
[STOLEN from Team Name] must name the exact team the idea came from — vague or missing tag = scored as regular, no ×3 multiplier
The improvement must be substantively better — Claude judges this at scoring time
If Claude determines it is not better: 0 score for that entry — no tier points, no multiplier
Maximum 3 steals per team — one per Steal Power Up Card
